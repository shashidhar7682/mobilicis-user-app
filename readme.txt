===>Prequestisites

Mongodb, MongoShell, Mongodb-Database-Tools, Nodejs, VSCode, Python.

===>Configuring MongoDB

->Open MongoShell (mongosh.exe) 
->Click Enter
->It Shows up all basic details of mongodb
->Now type "use test" command and click Enter
->Now type "db.createCollection("userCollection)" command and click Enter


===>Clone The Repository

-> use https://github.com/shashidhar7682/mobilicis-user-app.git to clone the repository
->and extract the zip if clone is zipped

===>Importing Sample Data into MongoDB - database "test" - collection "userCollection"

->open cmd and Type "pip3 install pymongo" to install pymongo module
->Now open The Repository in local directory using VSCode
->Now open import.py file in Python_Mongo_Importer and run the python file

===>Installing All neccessary modules to run the app

->Open terminal in root directory of app and run "npm install"
->Now To run Backend - install nodemon 
->Open terminal and run "npm install -g nodemon" 

===>Starting Backend Server

->In terminal in VSCode and run the command "nodemon server.js"

===>Starting Frontend App

->Open a new Terminal in VSCode and run the command "npm start"

===>Click On Tables in Navigation Bar to List the Users Based on Assignment Queries
